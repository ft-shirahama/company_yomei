# 資金繰りシミュレーション_バーンレート_余命_フォーマット.xlsx
本ファイルは、主に株式会社などの法人が資金繰り・キャッシュフローを計算したいときに使えるExcelファイルになります。
資金調達、借入金返済などにも対応し、あと何ヶ月で倒産するかの余命を算出できます。

# 使用方法
詳細な使い方は「サンプル」シートをご参照ください。
